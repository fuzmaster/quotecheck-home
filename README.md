# QuoteCheck Home

A one-page Vite + React + TypeScript tool that helps homeowners compare contractor quotes, spot missing scope details, and generate follow-up questions before signing.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

Stripe payment routes are included for Vercel serverless deployment. Development mode falls back to a local mock unlock if Stripe is not configured.
